API

It's one GET request.

No SDK to learn. No config to write. Just a URL you can paste into your browser.

Terminal — bash 
# Generate an OG image in one line
curl "https://snapog.dev/og\
  ?title=Hello+World\
  &description=Your+first+OG+image\
  &domain=mysite.com\
  &template=default\
  &theme=dark\
  &key=sk_YOUR_API_KEY" \
  -o og-image.png

# That's it. Drop it in your <head>:
<meta property="og:image" content="https://snapog.dev/og?title=..." />

Works with React, Next.js, Hugo, Astro, Svelte, PHP — anything that can make an HTTP request.
